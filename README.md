
## Cách dùng
Gom hết file ppt/pptx vào một thư mục, copy đường dẫn tới thư mục đó

Tạo thêm thư mục để lưu file pdf thành phẩm nữa, cũng copy đường dẫn tới thư mục đó

Mở file python lên và thay đường dẫn của 2 thư mục bên trên nào rồi chạy file.


## Cách hoạt động
Mở PowerPoint bằng COM, lấy danh sách file trong thư mục
Với mỗi file .ppt / .pptx:
- Mở file
- SaveAs PDF
- Đóng lại
